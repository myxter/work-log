## 踩坑记录

| 问题 | 原因 | 解决 |
|---|---|---|
| 有道 API 302 | 封了免费 JSON 接口 | 换 MyMemory |
| CSS 污染网页 | `* { margin:0 }` 注入所有页面 | 拆成 bubble.css + popup.css |
| 动画不播放 | CSS animation 渲染时序问题 | `animation-play-state: paused` → `running` |
| cmd 输入栏错位 | cmd.exe 不支持完整 ANSI | 换 Windows Terminal |
| cmd 拉伸闪退 | SIGWINCH 信号导致进程中断 | 换 Windows Terminal |

## 讨论的重要概念

| 话题 | 要点 |
|---|---|
| **油猴 vs 浏览器插件** | 插件适合独立功能，油猴适合快速注入网站 |
| **LLM 翻译 vs 翻译记忆库** | LLM 理解语境但慢，记忆库快但术语不准 |
| **RAG 知识库** | 检索增强生成：先查文档再回答，研究生读论文必备 |
| **Dify vs AnythingLLM** | Dify 功能更强需 Docker，AnythingLLM 零代码速开 |
| **LLM AIO Gateway** | API 统一网关，协议转换 + 模型路由，像网络路由器 |
| **Harness** | LLM 的编排层，CC 是重型 Harness，Agent Loop 是核心 |
| **MCP（Model Context Protocol）** | 让 Claude 直接操控外部服务（GitHub/数据库/Notion） |
| **Docker** | 应用集装箱，一键启动复杂环境 |
| **Markdown** | 纯文本排版，用 `#` `**` `-` 代替 HTML |
| **吴恩达 ML vs DL** | ML 传统算法（SVM/决策树），DL 神经网络/Transformer |
| **CC + DeepSeek V4** | 当前最优组合：稳定、快、省钱、缓存适配好 |
| **cmd.exe** | 1993年的终端，TUI 兼容性差，换 Windows Terminal |
