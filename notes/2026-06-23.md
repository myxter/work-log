# 会话记录 2026-06-23

## 完成事项

### 划词翻译浏览器插件 ✅
- 为 Edge 浏览器开发了第一个插件：选中英文 → 弹出中文翻译气泡
- 技术栈：Manifest V3 + HTML/CSS/JS
- 翻译服务：MyMemory API（免费，无需 Key）

## 项目结构

```
translate-extension/
├── manifest.json       ← 插件配置
├── background.js       ← Service Worker：翻译 API 代理
├── content.js          ← 划词检测 + 气泡显示
├── popup.html          ← 点击图标弹出的小窗口
├── popup.js            ← popup 逻辑（开关控制）
├── styles/
│   ├── bubble.css      ← 注入网页的气泡样式（#tt-bubble 隔离）
│   └── popup.css       ← popup 窗口样式（不注入网页）
└── icons/              ← 蓝色方块图标
```

## 踩坑记录

| 问题 | 原因 | 解决 |
|---|---|---|
| 有道 API 返回 302 | 有道封了无认证的 JSON 接口 | 换 MyMemory API |
| 网页布局错乱 | `* { margin:0 }` 随 CSS 注入所有页面 | 拆成 bubble.css（注入）+ popup.css（不注入） |

## 学到的知识点

- 浏览器插件架构：content script → background service worker 的消息传递
- CSS 隔离：用 `#id` 前缀避免影响宿主页面
- API 选型：国内环境需测试可访问性
- Edge 和 Chrome 插件完全兼容（都是 Chromium 内核）
