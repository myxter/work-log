# 会话记录 2026-06-25

## 完成事项

### 提示词缓存机制理解 ✅
- API prompt cache 是**前缀字节匹配**，不是内容相关性匹配
- 缓存 TTL 默认 5 分钟，连续对话中每次命中会重置倒计时
- CC 进程结束后缓存消失，重新打开算全新请求
- 长对话每次仍然全量发送历史，但绝大部分走缓存（~1/10 价格）
- 实测：两天后重新打开 CC，首次请求全部无缓存，花费 1.7 元

### 终端问题诊断 ✅
- cmd.exe 是 1993 年的终端，TUI 兼容性差导致输入栏错位
- SIGWINCH 信号导致拉伸窗口时 CC 闪退
- 推荐换 Windows Terminal（256 色、GPU 渲染、不闪退）
- `tui: fullscreen` 设置对 cmd 没效果
