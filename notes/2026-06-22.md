## 讲到的知识点

| 理论 | 代码体现 |
|---|---|
| DBMS / SQLite | `sqlite3.connect()` |
| SQL CRUD | INSERT / UPDATE / DELETE / SELECT |
| 事务 | `commit()` / `rollback()` |
| 外键关联 | `INNER JOIN ... ON p.ID = np.PERSON_ID` |
| 单例模式 | `ConnectionFactory` 和 `Crud` 类 |
| 抽象类 / 继承 | `Person(ABC)` → `NaturalPerson` |
| 封装 | `@property` getter/setter |
| SQL 注入防护 | `:NAME` 占位符 |
| Python import 机制 | `from services.crud import Crud` |
